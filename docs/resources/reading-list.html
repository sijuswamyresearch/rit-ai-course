Additional Reading & Learning Resources

Curated resources for deepening your AI & ML knowledge

A comprehensive 20-hour intensive course on Applied AI and Machine Learning designed for engineering students from all disciplines. Covers mathematical foundations, classical ML, deep learning, NLP with transformers, and deployment with Streamlit. Includes hands-on projects and capstone implementation.

Essential Books

Foundational Textbooks

  1. “Deep Learning” by Ian Goodfellow, Yoshua Bengio, and Aaron Courville
    • The definitive textbook on deep learning
    • Available free online at deeplearningbook.org
    • Covers mathematical foundations, modern practices, and research perspectives
  2. “Pattern Recognition and Machine Learning” by Christopher M. Bishop
    • Classic text on probabilistic approaches to ML
    • Excellent for understanding Bayesian methods
    • Published by Springer, 2006
  3. “Machine Learning: A Probabilistic Perspective” by Kevin P. Murphy
    • Comprehensive coverage of ML from probabilistic viewpoint
    • Includes MATLAB code examples
    • MIT Press, 2012

Practical Guides

  1. “Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow” by Aurélien Géron
    • Best practical guide for implementing ML in Python
    • Covers both classical ML and deep learning
    • O’Reilly, 3rd Edition (2022)
  2. “Introduction to Machine Learning with Python” by Andreas Müller and Sarah Guido
    • Beginner-friendly introduction
    • Focus on scikit-learn
    • O’Reilly, 2016

Mathematics for AI

  1. “Introduction to Linear Algebra” by Gilbert Strang
    • Excellent for understanding the linear algebra behind ML
    • MIT OpenCourseWare available
    • Wellesley-Cambridge Press, 5th Edition (2016)
  2. “Mathematics for Machine Learning” by Marc Peter Deisenroth, A. Aldo Faisal, and Cheng Soon Ong
    • Specifically designed for ML practitioners
    • Available free online at mml-book.com
    • Cambridge University Press, 2020

Online Courses

University Courses

  1. CS229: Machine Learning (Stanford)
    • Andrew Ng’s legendary course
    • Available on YouTube
    • Mathematical and theoretical focus
  2. CS231n: Convolutional Neural Networks for Visual Recognition (Stanford)
    • Best course for computer vision
    • Available at cs231n.stanford.edu
    • Includes programming assignments
  3. CS224n: Natural Language Processing with Deep Learning (Stanford)
    • Comprehensive NLP course
    • Available at web.stanford.edu/class/cs224n
    • Covers transformers, BERT, GPT

MOOC Platforms

  1. Deep Learning Specialization (Coursera - DeepLearning.AI)
    • 5-course specialization by Andrew Ng
    • Covers neural networks, CNNs, RNNs, transformers
    • Link
  2. Machine Learning Specialization (Coursera - Stanford/DeepLearning.AI)
    • Updated version of Andrew Ng’s original ML course
    • Uses Python instead of MATLAB
    • Link
  3. Fast.ai Practical Deep Learning for Coders
    • Top-down, code-first approach
    • Free course with excellent community
    • Link

Research Papers

Foundational Papers

  1. “Attention Is All You Need” (Vaswani et al., 2017)
    • The Transformer paper that revolutionized NLP
    • arXiv:1706.03762
  2. “ImageNet Classification with Deep Convolutional Neural Networks” (Krizhevsky et al., 2012)
    • AlexNet - the paper that started the deep learning revolution
    • arXiv:1203.5726
  3. “Deep Residual Learning for Image Recognition” (He et al., 2015)
    • ResNet - enabled training of very deep networks
    • arXiv:1512.03385
  4. “Generative Adversarial Nets” (Goodfellow et al., 2014)
    • Introduced GANs for generative modeling
    • arXiv:1406.2661
  5. “Adam: A Method for Stochastic Optimization” (Kingma & Ba, 2014)
    • The most popular optimizer for deep learning
    • arXiv:1412.6980

Recent Advances

  1. “Language Models are Few-Shot Learners” (Brown et al., 2020)
    • GPT-3 paper demonstrating emergent abilities
    • arXiv:2005.14165
  2. “BERT: Pre-training of Deep Bidirectional Transformers” (Devlin et al., 2018)
    • Bidirectional pre-training for NLP
    • arXiv:1810.04805

Recommended Learning Path

For Complete Beginners

  1. Start with 3Blue1Brown videos for mathematical intuition
  2. Take Andrew Ng’s ML Specialization on Coursera
  3. Read Géron’s Hands-On ML book
  4. Practice on Kaggle with beginner competitions

For Those with Programming Background

  1. Fast.ai course for practical deep learning
  2. CS231n or CS224n for specialized topics
  3. Read Goodfellow’s Deep Learning textbook
  4. Contribute to open-source projects on GitHub

For Advanced Practitioners

  1. Follow arXiv daily for new papers
  2. Implement papers from Papers with Code
  3. Participate in Kaggle competitions
  4. Contribute to research or open-source projects

Final Advice

“The best way to learn AI is by doing. Read a paper, implement it, break it, fix it, and then build something new.”

Remember: - Start simple and gradually increase complexity - Build projects that interest you personally - Join communities to learn from others - Stay curious and keep experimenting - Share your work - teaching others reinforces your learning

Happy Learning! 🚀
