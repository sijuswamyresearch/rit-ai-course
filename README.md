## 📖 Course Overview
This repository contains the complete curriculum, lecture slides, and study materials for the **20-Hour Intensive Foundation Course in AI & ML**, designed specifically for final-semester B.Tech students. 

The curriculum transitions engineering students from foundational mathematics (Linear Algebra, Calculus, Probability) to deploying modern Generative AI applications (RAG, LLMs, Streamlit). It is structured strictly within the parameters of the **Outcome-Based Education (OBE)** framework mandated by the NBA.

## 📚 Curriculum Structure

The course is divided into 5 intensive modules:

- **Module 1: Mathematical Foundations for AI** (Linear Algebra, Optimization, Probability)
- **Module 2: Classical Machine Learning & Data Workflows** (Scikit-Learn, Regression, Classification, Clustering)
- **Module 3: Deep Learning Architectures** (Neural Networks, Activation Functions, TensorFlow/Keras)
- **Module 4: NLP & Generative AI** (Transformers, Hugging Face, RAG with NotebookLM)
- **Module 5: Capstone Integration & Deployment** (ModelOps, Streamlit web apps)

*For detailed Course Outcomes (COs) and PO mapping, please refer to the [Syllabus](syllabus.qmd) document.*

## 🛠️ Technology Stack
Students will gain hands-on experience with the modern AI ecosystem:
- **Languages & Frameworks:** Python, Scikit-Learn, TensorFlow, Keras
- **NLP & GenAI:** Hugging Face Transformers, NotebookLM, LangChain concepts
- **Deployment:** Streamlit
- **Environments:** Google Colab, Jupyter Notebooks
- **Data & Versioning:** Kaggle API, Git, GitHub
